Самыми популярными форматами являются:

    1. CSV (англ. Comma-Separated Values - значения, разделенные запятыми)
    2. JSON (англ. JavaScript Object Notation) - текстовый формат обмена данными, основанный на JavaScript
    3. XML (англ. eXtensible Markup Language - расширяемый язык разметки)
    4. YAML (англ. YAML Ain’t Markup Language - «YAML - Не язык разметки»)
    5. INI (англ. Initialization file - файл инициализации)

Большинство форматов файлов поддерживается Python
